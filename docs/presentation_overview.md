# WarpDrive Pomodoro - アプリ概要と技術的特徴

## 1. プロジェクト概要
WarpDrive Pomodoro は、**ポモドーロ学習/作業**をベースに、
**チーム参加による加速体験**を組み合わせた Web アプリです。

- 1人で進めるだけでなく、メンバー招待で共同作業できる
- オンライン人数に応じて、目標到達（残り時間消化）が加速する
- 宇宙航行の演出で、進捗を直感的に可視化する

## 2. 解決したい課題
従来のタイマーアプリの課題:
- 継続モチベーションが落ちやすい
- チーム作業の「見える化」が弱い
- 進捗が数字だけで体感しづらい

本アプリのアプローチ:
- ゲーミフィケーション（宇宙航行UI + 目的地）
- 招待ベースのチーム参加
- オンライン人数連動の進行速度

## 3. 主な機能
### 3.1 認証・アクセス制御
- サインアップ / ログイン / ログアウト
- JWTをHTTP-only Cookieで保持
- フロント側のルート保護（未ログイン時は `/login` に強制遷移）

### 3.2 タスク作成・再参加
- 目標（惑星）を選んでタスク作成
- `goal` 画面右側に過去/参加タスク一覧を表示
- タスクをクリックして再参加

### 3.3 作業セッション管理
- `START` で作業開始（時刻記録）
- `STOP` で終了（作業時間集計）
- タスク全体の `total_time` / `leave_time` を更新

### 3.4 チーム招待・オンライン反映
- メールアドレスでメンバー招待
- メンバー一覧表示、オンライン状態表示
- オンライン人数を定期監視し、進行速度に反映

## 4. デモ時に見せる流れ（発表向け）
1. ログイン（未ログインなら保護ルートに入れないことを確認）
2. `goal` で目的地を選び「出発」
3. `app` で `START` → 距離/残り時間が進む
4. 右側からメンバー招待
5. 別端末で参加して `START`
6. オンライン人数増加と、目標時間の減少速度アップを確認
7. `STOP` でDBへ反映（累積時間更新）

## 5. システム構成
### フロントエンド
- Next.js 16 (App Router)
- React 19 + TypeScript
- カスタム宇宙UI（Canvas/アニメーション演出）

### バックエンド
- FastAPI
- SQLAlchemy ORM
- JWT認証（Cookieベース）

### データベース
- PostgreSQL
- 主要テーブル:
  - `users`: ユーザー情報
  - `tasks`: 目標時間・残り時間・オンライン人数
  - `members`: タスク参加者
  - `times`: セッション開始/終了・作業秒数

## 6. 技術的特徴（実装観点）
### 6.1 チーム連動型カウントダウン
- 基本は1秒ごとに進行
- `online_member_count` を乗数として残り時間を減算
- 例: オンライン2人なら 1秒で2秒分進む

### 6.2 セッション単位の正規化された時間管理
- 開始時に `times` レコードを生成
- 終了時に差分秒を確定
- `members.sum_time` と `tasks.total_time` を再集計
- `tasks.leave_time = goal_time - total_time` を保持

### 6.3 API分割による責務明確化
- `/auth/*`: 認証
- `/tasks/*`: タスク作成・参加者・招待・状態
- `/work/start`: セッション開始
- `/time/{time_id}`: セッション終了

### 6.4 UI側の運用安定化
- 定期ポーリングで状態同期（メンバー/ステータス）
- キャッシュ回避付き取得
- API返却の型ゆれ（キー差異）に対する防御的パース

## 7. 非機能面の工夫
- CORS設定でフロント/バックを分離開発
- ルートガードで誤アクセスを防止
- 画面演出と業務ロジックを分離し、保守しやすい構造

## 8. 今後の改善（発表での伸びしろ）
- WebSocket化によるリアルタイム同期（ポーリング削減）
- JWT秘密鍵の環境変数化 + セキュリティ強化
- 通知（招待受信・目標到達）
- 分析ダッシュボード（週間/月間生産性）

## 9. まとめ
WarpDrive Pomodoro は、
**「個人の時間管理」×「チーム連携」×「ゲーミフィケーション」**を統合した学習/作業支援アプリです。

単なるタイマーではなく、
**仲間と進むことで速度が上がる体験**を通して、継続的な集中と達成感を生み出します。
